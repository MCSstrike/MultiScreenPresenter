Trust Certificate For Screen Sharing

Use this HTTP page to download the local certificate, install it, then use the HTTPS site.

Step 1: Download Certificate

If download fails, start or restart Docker so the proxy can generate the certificate first.

Download multiscreen-caddy-root.crt

Step 2: Install On Your OS

Windows
  1. Open downloaded multiscreen-caddy-root.crt.
  2. Click Install Certificate.
  3. Select Local Machine (or Current User if preferred).
  4. Choose Place all certificates in the following store.
  5. Select Trusted Root Certification Authorities.
  6. Finish and accept the warning prompt.
  7. Close and reopen your browser.
macOS
  1. Open Keychain Access.
  2. Drag multiscreen-caddy-root.crt into System keychain (or Login keychain).
  3. Double-click the imported certificate.
  4. Expand Trust and set When using this certificate to Always Trust.
  5. Enter your password if prompted.
  6. Close and reopen your browser.
Linux

Linux steps vary by distro and browser. Typical Debian/Ubuntu flow:

  1. Copy cert to /usr/local/share/ca-certificates/ as .crt.
  2. Run sudo update-ca-certificates.
  3. Restart browser.

For Firefox, import the cert in browser certificate settings if needed.

Step 3: Open Secure Site

After cert trust is installed, open:

  • https://YOUR_HTTPS_HOST/control
  • https://YOUR_HTTPS_HOST/display?displayId=display-1
